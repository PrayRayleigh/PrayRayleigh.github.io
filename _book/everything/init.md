# 第一次搭建博客
## 原由
  &emsp;&emsp;这几天老下雨,衣服也是没干。这学期的考试也考完了，明知道还有不少事情要做，但是我好像一点动力没有。今天早上收到一封邮件说我的域名快过期了，才想起来我去年买了个域名，本来准备做一个博客的。但是一直搁置，所以今天下午花了一个下午的时间把它做出来了。也算是能有个记录自己想法的空间，给以后的自己看或者给朋友看也是不错的。
## 吐槽
  &emsp;&emsp;两年没碰Git，早就忘光光了。以前就是简单的pull push，这次弄博客算是更深地学习了一下，对我这个小白也有益处。还有一点就是Gitbook对markdown的支持不能说非常完善吧，也可以说是完全没有。也让我这个从来没碰过这方面的人见识了一下。现在就是在main分支上写，再把它挂在pages分支上。
  <br/>
  &emsp;&emsp;据说gitbook出了新版，ui更加漂亮，不过我是懒得折腾了。
  <br/>
  &emsp;&emsp;挺好！希望自己以后能多在这个地方记录吧，能静下心来写写东西总归是好的。
